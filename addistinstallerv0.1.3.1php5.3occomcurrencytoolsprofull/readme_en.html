Currency Tools PRO

Main features:
- Automatically detects the visitor's country and sets the defined currency depending on the settings.
- Adds the ability to choose for each product a different currency.
- Adds the ability to choose for each manufacturer separate currency.
- Adds the ability to assign to each manufacturer their own currency rates.
- Adds the ability to choose margin for each product (as a percentage and a fixed amount in a single currency).
- Adds the ability to choose different currency(on checkout), different margin and different overheads(fixed amount) for each country.
- Adds the ability to currency conversion products by brand, by category and other criterias.
- Adds the ability to filter payment methods in currencies.

Other features:
- Adds the ability to choose to different admin currency.
- Adds the ability to choose a different default currency for site.
- Adds the ability to choose a secondary (comparative) currency (currency in scopes).
- Adds the ability to specify a different currency for payment.
- Adds the ability to specify different currency for each payment method.
- Adds the ability to sespecify different source currency for each delivery method.
- Adds the ability to set a negative value for the rounding of prices for currencies.
- Adds the ability to set the value for the correction rate for each currency separately.
- Adds the ability to set margins for each currency separately.
- Adds the ability to set a fixed premium for each currency separately.
- Adds the ability to select the source of the currency rate.
- Adds the ability to assign the clock automatically update currency rates.
- Adds the ability to exclude a particular currency from the update rate.
- Adds the ability to configure / edit the widget informer.
- Adds the ability to customize the widget informer for each language.
- Adds the ability to set different settings for shops (support multi-store).

Instructions:
1. Installation:
- Download the archive compatible with your php version.
- Upload the contents of the upload folder to the root of your site.
- Install the Addist Extension Installer on the admin by Modules » Addist Extension Installer » [Install].
- Open the Addist Extension Installer, authenticate, find the purchased extension and download it.
- Open the extensions page, find the downloaded extension and install it.
2. Manual update:
- Upload the contents of the archive in the root of the site.
- Open the extension, open the tab "Other Settings", click on the "Fix" button.
- Clean and refresh the modifications cache (if Opencart 2.x).
3. Update via the Update Center:
- Open the module, open the tab "Other Settings", click on the "Check for updates".
- Click on "Update" in the message.
- Clean and refresh the modifications cache (if Opencart 2.x).
