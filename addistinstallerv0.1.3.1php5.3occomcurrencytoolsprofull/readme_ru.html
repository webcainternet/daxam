Валютный Менеджер PRO

Особенности:
- автоматически определяет страну посетителя и устанавливает нужную валюту в зависимости от настроек.
- добавляет возможность выбрать для каждого товара отдельную валюту.
- добавляет возможность выбрать для каждого производителя отдельную валюту.
- добавляет возможность назначить для каждого производителя свои курсы.
- добавляет возможность выбрать для каждого товара наценки(в процентах и фиксированную сумму в отдельной валюте).
- добавляет возможность назначить разные валюты(при заказе), разные наценки, разные надбавки для разных стран.
- добавляет возможность конвертация валюты товаров по брендам, по категориям и по другим критериям.
- добавляет возможность фильтрации способов оплаты по валютам.

Прочие возможности:
- добавляет возможность выбрать для админки отдельную валюту.
- добавляет возможность выбрать для сайта отдельную валюту по умолчанию.
- добавляет возможность выбрать вторичную(сравнительную) валюту(валюта в скобках).
- добавляет возможность установки для оплаты отдельную валюту.
- добавляет возможность установки для каждого способа оплаты отдельную валюту.
- добавляет возможность установки для каждого способа доставки отдельную исходную валюту.
- добавляет возможность установки отрицательного значения для округления цен для валют.
- добавляет возможность установки значения для коррекции курса для каждой валюты отдельно.
- добавляет возможность установки наценки для каждой валюты отдельно.
- добавляет возможность установки фиксированную надбавку для каждой валюты отдельно.
- добавляет возможность выбора источника курса валют.
- добавляет возможность назначать часы автоматического обновления курса валют.
- добавляет возможность исключения конкретной валюты из обновления курсов.
- добавляет возможность настроить/редактировать вида виджета-информера.
- добавляет возможность настройки виджета-информера для каждого языка отдельным шаблоном.
- добавляет возможность устанавливать разные настройки для магазинов(поддержка мульти-магазина).

Лицензия:
- Данный продукт может не работать на локальном сайте.
- Лицензия действует только на один домен + на все домены 3го уровня(на под-домены).
- Чтобы установить данный продукт на другом домене, необходима приобрести продукт заново.
- Чтобы сменить домен, достаточно написать на имеил dev@addist.ru.
- Для активации нет необходимости связаться со мной, данные о заказе поступают на сервер лицензирования моментально, для активации достаточно ввести данные покупки в соответствующие поля и продукт активируется моментально.

Инструкция:
1. Установка:
- Загрузить подходящий для вашей версии php архив.
- Загрузить содержимое папки upload в корень сайта.
- Выполнить установку установщика дополнений в админке через Дополнения » Модули » Установщик дополнений Addist » [Установить].
- Открыть модуль Установщик дополнений Addist, авторизоваться и загрузить купленный модуль из списка.
- Открыть список дополнений и установить загруженный модуль.
2. Ручное обновление:
- Загрузить содержимое архива в корень сайта.
- Открыть модуль, открыть вкладку "Прочие настройки", нажать на кнопку "Исправить/Переустановить".
- Очистить и обновить кэш модификаций(если Opencart 2.x).
3. Обновление через центра обновлений:
- Открыть модуль, открыть вкладку "Прочие настройки", нажать на кнопку "Проверить обновления".
- Нажать на ссылку "Обновить" в появившиеся сообщения.
- Очистить и обновить кэш модификаций(если Opencart 2.x).
